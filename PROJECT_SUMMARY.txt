PROJECT OVERVIEW
================================================================================
A comprehensive Discord moderation bot with a full-featured web dashboard for
server management. The project consists of three main components: Discord Bot,
Backend API, and React Dashboard.

TECH STACK
================================================================================
Bot:
  - Discord.js v14.25.1
  - MongoDB (Mongoose v8.8.3)
  - Node.js with dotenv

Backend API:
  - Express.js v4.21.1
  - MongoDB (Mongoose v8.8.3)
  - JWT Authentication
  - Discord OAuth2
  - Axios for Discord API calls
  - CORS & Rate Limiting

Dashboard (Frontend):
  - React v18.3.1
  - React Router v6.28.0
  - Chart.js v4.5.1 (Analytics)
  - React Hot Toast (Notifications)
  - React Icons v5.5.0
  - Axios for API calls

KEY FEATURES
================================================================================

MODERATION COMMANDS (8)
--------------------------------------------------------------------------------
✅ /ban          - Ban users with reason logging
✅ /kick         - Kick users with reason logging
✅ /mute         - Mute users with role-based muting
✅ /unmute       - Unmute users
✅ /warn         - Warn users (tracked in database)
✅ /timeout      - Timeout users (Discord native)
✅ /purge        - Bulk delete messages (up to 100)
✅ /slowmode     - Set channel slowmode

AUTOMODERATION
--------------------------------------------------------------------------------
✅ Spam Detection     - Auto-detect message spam
✅ Link Detection     - Block/log unauthorized links
✅ Mass Ping Detection - Prevent mass mentions
✅ Word Filter        - Custom word filtering

LOGGING SYSTEM (5 Types)
--------------------------------------------------------------------------------
✅ Moderation Logs    - All mod actions logged with case IDs
✅ Message Logs       - Deleted/edited messages
✅ Member Logs        - Joins/leaves/updates
✅ Voice Logs         - Voice channel activity
✅ Role Logs          - Role changes

UTILITY FEATURES
--------------------------------------------------------------------------------
✅ Ticket System      - Support ticket channels
✅ Reaction Roles     - Role assignment via reactions
✅ Custom Commands    - User-defined commands with embeds
✅ Announcements      - Schedule server announcements
✅ Reminders          - Personal user reminders

DASHBOARD FEATURES
--------------------------------------------------------------------------------
✅ Discord OAuth      - Secure login with Discord

API ROUTES
================================================================================

Authentication:
  POST   /api/auth/discord        - Discord OAuth login
  POST   /api/auth/callback       - OAuth callback
  GET    /api/auth/me             - Get current user
  POST   /api/auth/logout         - Logout user

Server Management:
  GET    /api/servers              - Get user's servers
  GET    /api/server/:id           - Get server details
  GET    /api/server/:id/channels  - Get server channels
  GET    /api/server/:id/roles     - Get server roles
  GET    /api/server/:id/members   - Get server members
  GET    /api/server/:id/settings  - Get server settings
  PUT    /api/server/:id/settings  - Update server settings
  POST   /api/server/:id/embed/send - Send embed to channel

Feature Routes:
  GET    /api/features/logs/moderation/:id    - Moderation logs
  GET    /api/features/logs/messages/:id      - Message logs
  GET    /api/features/logs/members/:id       - Member logs
  GET    /api/features/logs/voice/:id         - Voice logs
  GET    /api/features/logs/roles/:id         - Role logs
  
  GET    /api/features/tickets/:id            - Get tickets
  POST   /api/features/tickets/:id            - Create ticket
  
  GET    /api/features/reactionroles/:id      - Get reaction roles
  POST   /api/features/reactionroles/:id      - Create reaction role
  DELETE /api/features/reactionroles/:id/:rrid - Delete reaction role
  
  GET    /api/features/customcommands/:id     - Get custom commands
  POST   /api/features/customcommands/:id     - Create command
  DELETE /api/features/customcommands/:id/:cid - Delete command
  
  GET    /api/features/announcements/:id      - Get announcements
  POST   /api/features/announcements/:id      - Create announcement
  DELETE /api/features/announcements/:id/:aid - Delete announcement
